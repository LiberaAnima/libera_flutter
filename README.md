## Git Rules

### Prefixes

```
fix: バグ等の修正
feat: 新機能
refactor: 動作に影響のないコード変更
docs: ドキュメントのみの変更
style: スタイルのみの変更
perf: パフォーマンス向上関連の変更
chore: ライブラリ等の変更
test: テスト関連の変更
**!: 破壊的変更
```

### Branch Name

```
ex) {your_name}/feat_***
```

### Commit Message

```
ex) feat: xxxx
```
